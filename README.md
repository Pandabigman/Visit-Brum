# React + Vite

This is a react vite project to demonstrate a hi-fi weekend city break website focusing on birmingham. 

An advanced feature will be implemented and tested for usability etc. 

## Run instruction
  In your terminal run the following commands sequentially
  - npm install
  - npm run dev
  - ctrl + O if using vs code to open the page/ or just click on the pop up link 
  - the homepage should be visible on success at 'http://localhost:5173/'.
